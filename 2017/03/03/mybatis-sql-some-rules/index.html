Mybatis中sql的大于、小于等符号的写法

发表于 2017-03-03   |   分类于 笔记   |  

mybatis中sql的大于、小于等符号的写法，

其实就是xml的特殊符号，因为它的配置就是xml，所以可以用下面这种写法转义

&lt;          < 
&gt;          >  
&lt;&gt;   <>
&amp;      & 
&apos;      '
&quot;      "

也可以使用<![CDATA[ ]]>符号进行说明，将此类符号不进行解析
